🔒 Security & Privacy
Keep SensoryWave private — only your family can access it
🔢 PIN Lock
Lock app with a 4-digit PINFamily members will need to enter it to open the app
Set / Change PIN
How it works:
When PIN lock is on, anyone who opens the app will see a lock screen asking for your PIN. Only people who know the PIN can get in.

💡 Tips:
• Choose a PIN your family all know
• If you forget your PIN, you can reset it from this screen
• The app still works offline — no internet needed
• Your Medical ID is stored only on this device — never online

💾 Backup & Transfer
Save all your data as a file — restore it on any device
📤 Save Everything to a File
Downloads a backup file with your Medical ID, PIN, language choice and travel settings. Send this file to yourself by email or save it to iCloud / Google Drive.
⬇️ Download My Backup File
📥 Restore From a File
On your new device, tap below and select the backup file. Everything will be restored instantly — Medical ID, PIN, language, travel settings.
⬆️ Load Backup File
🗑️ Clear All My Data
Removes everything from this device — Medical ID, PIN, language and travel settings. Use this if you are giving the device to someone else.
🗑️ Clear All Data From This Device
Moving to a new phone?
1. On your old phone → tap "Download My Backup File"
2. Email it to yourself or save to cloud
3. On new phone → open SensoryWave → Backup tab → "Load Backup File"
4. Everything is restored ✅

Works iPhone ↔ Android — the file works on any device.
